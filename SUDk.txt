## SUD規格の定義

任意の多層系 `X = {x_0, x_1, ..., x_N}` を受け取り、φ正規化座標 `r` に変換する。

```python
phi = (1 + 5**0.5) / 2

# ステップ1: φスケールの設定
low[i]  = phi ** i          # 層iの下限
span[i] = phi ** i * (phi - 1)  # 層iの幅

# ステップ2: 正規化
r[i] = (x[i] - low[i]) / span[i]  # φ空間での位置
```

これだけ。

正規化後は **SUD動力学（タワー+ウロボロス）** がそのまま動く。

## コード

```python
phi = (1 + 5**0.5) / 2

def to_sud_space(x_array, N=8):
    """任意の多層配列をSUD規格に変換"""
    low  = [phi**i for i in range(N)]
    span = [phi**i * (phi-1) for i in range(N)]
    r    = [(x_array[i] - low[i]) / span[i] for i in range(N)]
    return r

def sud_attractor_target():
    """SUD規格のアトラクタ目標値"""
    return 1 / phi  # 0.6180...
```
