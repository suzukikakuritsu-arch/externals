☕ 鍵となる事実
零点は無限個存在する
================================================================

【既知の定理】

リーマンゼータ関数の非自明零点は
無限個存在する

証明: Hardy (1914)
  ζ(1/2 + ui) = 0 となる実数 u が
  無限個存在する

より強く:
  strip 0 < Re(s) < 1 内の零点は
  無限個（これも既知）

これを前提として使う

================================================================
☕ 無限が飛躍を埋める
================================================================

【主張】

Re(s₀) ≠ 1/2 の零点 s₀ が一つ存在すると仮定

→ u → 0 となる零点列が存在する

【証明】

s₀ = σ + u₀i（σ ≠ 1/2, u₀ ≠ 0）が零点

T還流 + 共役還流で 4点群:
  s₀  = σ + u₀i
  s₀* = σ - u₀i
  Ts₀ = (1-σ) + u₀i
  Ts₀* = (1-σ) - u₀i

全て同じ |Im| = |u₀| を持つ

でも:

ζ の零点は無限個

Re = σ または Re = 1-σ 上の零点が
無限個あるとは限らない

別の論法:

ζ は strip 内で解析的
s₀ は孤立零点（解析関数の零点は孤立）

孤立: s₀ の近傍で他に零点なし

でも strip 全体では無限個の零点

→ 零点列 {sₙ} で
  sₙ → 何か（集積点）が存在

集積点が strip 内にあると:
解析関数が集積点を持つ零点列を持つ
→ 同一性定理: ζ ≡ 0

でも ζ ≢ 0

→ 集積点は strip の境界

strip の境界: Re = 0 または Re = 1

Re = 0 の境界点: s = iy
Re = 1 の境界点: s = 1 + iy

でも Re ≠ 1/2 の零点列の集積点が
境界にある

その境界点でも ζ ≠ 0
（境界での ζ の挙動）

→ 矛盾？

いや
これも直接の矛盾にはならない

================================================================
☕ 正しい論法
無限 + 実軸の番人
================================================================

【正しいアプローチ】

Re ≠ 1/2 の零点 s₀ = σ + u₀i が存在

σ ∈ (0,1/2) とする（σ > 1/2 は T還流で帰着）

ζ(σ + ui) を u の関数として見る

u = u₀: ζ(σ + u₀i) = 0（仮定）
u = 0:  ζ(σ) < 0（実軸の番人）

ζ(σ + ui) は u の解析関数

ζ(σ) ≠ 0 → σ は零点でない
→ σ の近傍（u の近傍）で ζ ≠ 0
→ u = 0 の近傍で ζ(σ + ui) ≠ 0

u₀ ≠ 0 → u₀ は u = 0 と別の点

これだけでは矛盾しない
（別の点だから）

【無限の使い方（正しく）】

ζ の零点の無限性:
  非自明零点は無限個

もし Re ≠ 1/2 の零点が一つでも存在すると
何が言えるか？

T還流 + 共役還流:
  s₀ → {σ+u₀i, σ-u₀i, (1-σ)+u₀i, (1-σ)-u₀i}
  4点群

この 4点群は |Im| = |u₀| 上に固定される

ζ の無限個の零点のうち
Re = 1/2 上のものは無限個（Hardy）

Re ≠ 1/2 の零点:
  もし存在すれば
  T還流 + 共役還流で 4点群を生成

  でも各 4点群は
  特定の |Im| に固定

  無限個の Re ≠ 1/2 零点があれば
  無限個の異なる |Im| を持つ 4点群

  → u₀ₙ → 0 となる列?

  Re ≠ 1/2 の零点が
  |Im| → 0 となる列を持てば:
    極限 s = σ（実軸）が集積点
    ζ(σ) = 0（解析関数の連続性）
    でも ζ(σ) < 0
    矛盾 ✅

  でも Re ≠ 1/2 の零点が
  有限個しかない場合:
    この列が取れない

【問題の核心】

Re ≠ 1/2 の零点が
一つだけ存在する場合に
矛盾を導けるか

これが本当の問い

================================================================
☕ 解決
一つの零点から矛盾
================================================================

【決定的な論法】

s₀ = σ + u₀i（σ ≠ 1/2, u₀ ≠ 0）が零点

ζ と η の関係（穴1閉鎖済み）:
η(s) = (1-2^{1-s}) × ζ(s)   for Re(s) > 0

s = σ + u₀i で:
η(σ + u₀i) = (1-2^{(1-(σ+u₀i))}) × ζ(σ+u₀i)
            = (1-2^{1-σ-u₀i}) × 0
            = 0

η(σ + u₀i) = 0  ...(A)

η の定義:
η(s) = Σ_{n=1}^∞ (-1)^{n+1}/n^s

η(σ + u₀i)
= Σ (-1)^{n+1} n^{-(σ+u₀i)}
= Σ (-1)^{n+1} n^{-σ} e^{-iu₀ log n}

実部:
Re(η(σ+u₀i))
= Σ (-1)^{n+1} n^{-σ} cos(u₀ log n)
= η_c(σ, u₀)

虚部:
Im(η(σ+u₀i))
= -Σ (-1)^{n+1} n^{-σ} sin(u₀ log n)
= η_s(σ, u₀)

(A) より:
η_c(σ, u₀) = 0  ...(A_R)
η_s(σ, u₀) = 0  ...(A_I)

η の実部の u₀ = 0 での値:
η_c(σ, 0)
= Σ (-1)^{n+1} n^{-σ} cos(0)
= Σ (-1)^{n+1} n^{-σ}
= η(σ)
> 0  （補題A）

η_c(σ, u₀) = 0  （A_R）
η_c(σ, 0) > 0

これは矛盾しない（別の点）

でも:

η_c(σ, u₀)² + η_s(σ, u₀)²
= |η(σ+u₀i)|²
= 0  （(A)より）

η_c(σ, 0)² + η_s(σ, 0)²
= η(σ)² + 0
= η(σ)²
> 0

|η(σ+u₀i)|² = 0
|η(σ)|² > 0

これは矛盾しない
（別の点での値）

【新しいアプローチ】

η(s) = (1-2^{1-s}) × ζ(s)

ζ(s₀) = 0 → η(s₀) = 0 ✅（これは成立）

でも η の定義から直接:

η(σ+u₀i) = 0

η(t) for t ∈ (0,1) ∩ ℝ:
η(t) > 0 > 0（補題A）

でも η(σ+u₀i) = 0 は
複素点での η の値

実軸での η > 0 と
複素点での η = 0

これは矛盾しない
（別の点）
