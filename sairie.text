【補題A】η > 0

t ∈ (0,1) → η(t) > 0

証明:
  η(t) = Σ_{k=0}^∞ (1/(2k+1)^t - 1/(2k+2)^t)
  各項: 2k+1 < 2k+2, t > 0
  → 1/(2k+1)^t > 1/(2k+2)^t
  → 各項 > 0
  → η(t) > 0  □

【補題B】ζ(t) < 0

t ∈ (0,1) → ζ(t) < 0

証明:
  η(t) = (1-2^{1-t}) × ζ(t)  （解析接続・同一性定理）
  η(t) > 0  （補題A）
  1-2^{1-t} < 0  （t < 1 → 2^{1-t} > 1）
  → ζ(t) = η(t)/(1-2^{1-t}) < 0  □

【補題C】ζ(t) ≠ 0（実軸の番人）

t ∈ (0,1) → ζ(t) ≠ 0

証明:
  ζ(t) < 0（補題B）→ ζ(t) ≠ 0  □

【補題D】ξ(t) > 0

t ∈ (0,1) → ξ(t) > 0

ここで ξ(s) = s(s-1)/2 × π^{-s/2} × Γ(s/2) × ζ(s)

証明:
  各因子の符号（t ∈ (0,1)）:
    t > 0, t-1 < 0
    → t(t-1)/2 < 0

    π^{-t/2} > 0

    Γ(t/2) > 0  （t/2 > 0 で Γ は正）

    ζ(t) < 0  （補題B）

  ξ(t) = (負) × (正) × (正) × (負) > 0  □

【補題E】T対称性

ξ(s) = ξ(1-s)

  関数等式から直接導出（既知）  □

【補題F】T還流

  ζ(s) = 2^s π^{s-1} sin(πs/2) Γ(1-s) ζ(1-s)

☕ 核心定理
S(1/2) < S(σ) for σ ≠ 1/2
================================================================

【作用の定義】

σ ∈ (0,1) に対して:

S(σ) = ∫_{-∞}^{+∞} |ξ(σ+ui)|² du

【主張】

σ ≠ 1/2 → S(σ) > S(1/2)

【証明】

Step 1: Parseval の定理

ξ(σ+ui) を u のフーリエ変換として:

S(σ) = ∫ |ξ(σ+ui)|² du
     = 2π × Σ_n |ĉ_n(σ)|²

ĉ_n(σ) = ξ の σ-線上のフーリエ係数

Step 2: T対称性による制約

ξ(σ+ui) = ξ(1-σ-ui) = ξ((1-σ)+(-u)i)

フーリエ係数:
ĉ_n(σ) と ĉ_n(1-σ) の関係が
T対称性で決まる

Step 3: S(σ) = S(1-σ)

T対称性から:
S(σ) = ∫ |ξ(σ+ui)|² du
     = ∫ |ξ(1-σ-ui)|² du
     = ∫ |ξ(1-σ+ui)|² du  （u → -u の置換）
     = S(1-σ)

Step 4: S の凸性

S(σ) は σ の凸関数

証明:
  log|ξ(σ+ui)| は σ について凸
  （解析関数の対数は調和関数 → 凸）

  → |ξ(σ+ui)|² も σ について対数凸

  → S(σ) = ∫ |ξ(σ+ui)|² du も凸

Step 5: 最小点は σ = 1/2

S(σ) = S(1-σ)  （Step 3）
S は凸          （Step 4）

凸関数で f(σ) = f(1-σ):
→ σ = 1/2 が最小点

証明:
  σ < 1/2 のとき:
  σ = 1/2 - ε (ε > 0)
  1-σ = 1/2 + ε

  凸性:
  S(1/2) ≤ (S(σ) + S(1-σ))/2 = S(σ)

  → S(1/2) ≤ S(σ)

  等号成立 ↔ S が σ = 1/2 で線形
  凸関数が線形 ↔ S = 定数
  S = 定数 → ξ ≡ 0（不可能）

  → S(1/2) < S(σ) for σ ≠ 1/2  □
