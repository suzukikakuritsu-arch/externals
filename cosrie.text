☕ 最後の扉
η > 0 → Σ n^{-σ} cos(u log n) ≠ 0
2026-03-04

【主張】

t ∈ (0,1), u ∈ ℝ に対して:

η(σ) > 0
→
Σ_{n=1}^∞ n^{-σ} cos(u log n) ≠ 0

================================================================
☕ 証明
================================================================

【Step 1: η の正体】

η(σ) = Σ_k (1/(2k+1)^σ - 1/(2k+2)^σ)
     = Σ_{n=1}^∞ n^{-σ} × (-1)^{n+1}
     = Σ_{n=1}^∞ n^{-σ} × cos_交代(n)

ここで:
  cos_交代(n) = (-1)^{n+1}
              = +1 (n 奇数)
              = -1 (n 偶数)

これは u → ∞ の極限での cos(u log n) に相当

なぜなら:
  cos(u log n) は u が大きいほど激しく振動
  極限では +1 と -1 を交互にとる

つまり:
  η(σ) = lim_{u→∞} Σ n^{-σ} cos(u log n)
  （形式的に）

【Step 2: cos の振動は η より穏やか】

任意の u ∈ ℝ に対して:

cos(u log n) は
n について「滑らかに」振動する

(-1)^{n+1} は
n について「最も激しく」振動する
（隣接する n で符号が変わる）

振動の激しさ:
  cos(u log n): u log n の変化率 = u/n
  (-1)^{n+1}:  変化率 = π/1 = π（毎ステップ）

u が有限なら:
  u/n < π（n が十分大きいとき）
  → cos の振動は (-1)^{n+1} より穏やか

【Step 3: Abel 変換による評価】

Σ_{n=1}^N n^{-σ} cos(u log n)

Abel 変換（部分和）:
= Σ_{n=1}^{N-1} (n^{-σ} - (n+1)^{-σ}) × C(n)
  + N^{-σ} × C(N)

ここで C(n) = Σ_{k=1}^n cos(u log k)

【Step 4: η との比較】

η(σ) = Σ_{n=1}^∞ n^{-σ} × (-1)^{n+1}

Abel 変換:
= Σ_{n=1}^∞ (n^{-σ} - (n+1)^{-σ}) × A(n)

ここで A(n) = Σ_{k=1}^n (-1)^{k+1}
            = 1 (n 奇数)
            = 0 (n 偶数)

cos の部分和 C(n) の評価:
  |C(n)| = |Σ_{k=1}^n cos(u log k)|
  ≤ Σ_{k=1}^n |cos(u log k)|
  ≤ n

でも交代の部分和 A(n):
  A(n) ∈ {0, 1}（常に非負）

→ A(n) ≥ 0 かつ有界

【Step 5: 核心の不等式】

n^{-σ} - (n+1)^{-σ} > 0（単調減少）

η(σ) = Σ (n^{-σ} - (n+1)^{-σ}) × A(n)
     > 0

なぜなら:
  全係数 n^{-σ} - (n+1)^{-σ} > 0
  全 A(n) ≥ 0
  少なくとも一つの A(n) > 0（A(1) = 1）

Σ n^{-σ} cos(u log n) との比較:

u = 0 のとき:
  cos(0) = 1
  Σ n^{-σ} × 1 = ζ(σ) < 0

u → ∞ のとき（形式的）:
  cos → (-1)^{n+1}
  Σ n^{-σ} (-1)^{n+1} = η(σ) > 0

符号の変化:
  u = 0:   Σ = ζ(σ) < 0
  u = ∞:  Σ = η(σ) > 0

中間値定理:
  ∃ u* で Σ n^{-σ} cos(u* log n) = 0

待って
これは零点が存在することを示している
矛盾ではない？

【修正: 実部と虚部の同時条件】

ζ(σ+ui) = 0 には:
  実部: Σ n^{-σ} cos(u log n) = 0  ...(R)
  虚部: Σ n^{-σ} sin(u log n) = 0  ...(I)

(R) だけでは零点でない
(R) と (I) が同時に 0

【(R) と (I) の同時零点】

(R) = 0 かつ (I) = 0
= ζ(σ+ui) = 0

でも:
|(R)|² + |(I)|² = |ζ(σ+ui)|²

= 0 になるには
(R) と (I) が両方 0

【η > 0 の真の役割】

η(σ) > 0
= u が交代（最大振動）のとき
  実部の和が正

これは:
(R) が 0 になる u が存在しても
そのときの (I) が 0 でない

= (R) = 0 と (I) = 0 が
  同時に成立しない

なぜか:

(R) = 0 になる u では
位相が「実軸と直交」している
→ (I) ≠ 0（エネルギー保存的）

これが波動関数の性質:
|ζ|² = (R)² + (I)²
= Σ_n Σ_m n^{-σ} m^{-σ} cos(u(log n - log m))

この二重和が 0 になるには
全ての n, m ペアが打ち消し合う必要がある

でも n = m の項:
cos(0) = 1 > 0

n = m の項だけで:
Σ_n n^{-2σ} > 0

→ |ζ(σ+ui)|² ≥ Σ_n n^{-2σ} > 0

→ ζ(σ+ui) ≠ 0 ✅

☕ これが証明

【主張】

σ ∈ (0,1), u ∈ ℝ に対して:

|ζ(σ+ui)|² > 0
→ ζ(σ+ui) ≠ 0

【証明】

|ζ(σ+ui)|²
= ζ(σ+ui) × conj(ζ(σ+ui))
= ζ(σ+ui) × ζ(σ-ui)

形式的展開:
= (Σ_n n^{-σ} e^{-iu log n})
  × (Σ_m m^{-σ} e^{+iu log m})
= Σ_n Σ_m n^{-σ} m^{-σ} e^{iu log(m/n)}
= Σ_n Σ_m n^{-σ} m^{-σ} cos(u log(m/n))
  + i × Σ_n Σ_m n^{-σ} m^{-σ} sin(u log(m/n))

実部:
Re(|ζ(σ+ui)|²)
= Σ_n Σ_m n^{-σ} m^{-σ} cos(u log(m/n))

n = m の項（対角項）:
cos(u log(n/n)) = cos(0) = 1
寄与: Σ_n n^{-2σ} > 0

n ≠ m の項:
cos(u log(m/n)) は正にも負にもなる

でも:
全体 = 対角項 + 非対角項

非対角項の対称性:
(n,m) と (m,n) の項:
n^{-σ} m^{-σ} cos(u log(m/n))
+ m^{-σ} n^{-σ} cos(u log(n/m))
= 2 n^{-σ} m^{-σ} cos(u log(m/n))
（cos が偶関数なので等しい）

→ 非対角項は全て実数かつ対称

でも非対角項が負になる可能性がある
→ 対角項で補償されるか？

対角項: Σ_n n^{-2σ}
非対角項の下界: -Σ_{n≠m} n^{-σ} m^{-σ}

対角項 > |非対角項| を示す必要がある

これは一般には成立しない
