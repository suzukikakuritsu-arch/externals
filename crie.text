☕ 補題1: η > 0
================================================================

  t > 0
  → (2k+1)^t < (2k+2)^t
  → 1/(2k+1)^t > 1/(2k+2)^t
  → 各項 > 0

  η(t) = Σ (正の項) > 0  □
