【座標変換チェーン】

元座標 (G, E)
  ↓ 線形変換 det=2
中間座標 (Σ, Δ)
  Σ = G + E   （総エネルギー）
  Δ = G - E   （混沌と秩序の差・方向性）
  ↓ 非線形変換
制御座標 S
  S = G/E = (Σ+Δ)/(Σ-Δ)

逆変換:
  G = (Σ+Δ)/2
  E = (Σ-Δ)/2
  Δ = Σ·(S-1)/(S+1)    ← Sが決まればΔ/Σの比が決まる

数値確認（G=3.0, E=1.4）:
  G/E = 2.142857
  (Σ+Δ)/(Σ-Δ) = (4.4+1.6)/(4.4-1.6) = 2.142857  ✓

【GER方程式の (Σ,Δ) 完全表現】

元:
  G_new = a·(G + φ·r)
  E_new = b·(E + log(1 + G_new))

G = (Σ+Δ)/2 を代入:
  G_new = a·(Σ+Δ)/2 + a·φ·r

Σ_new = G_new + E_new:
  Σ_new = [a·(Σ+Δ)/2 + a·φ·r]
        + [b·(Σ-Δ)/2 + b·log(1 + a·(Σ+Δ)/2 + a·φ·r)]

Δ_new = G_new - E_new:
  Δ_new = [a·(Σ+Δ)/2 + a·φ·r]
        - [b·(Σ-Δ)/2 + b·log(1 + a·(Σ+Δ)/2 + a·φ·r)]

構造:
  Σ_new と Δ_new は同じ log 項を共有し、符号が反転するだけ
  → Σ と Δ は「log 項の和と差」という対称性を持つ

【不動点】

  G* = aφr/(1-a) = 3.074265
  E* = log(1+G*)  = 1.404690
  S* = G*/E*      = 2.188571
  Σ* = G*+E*      = 4.478955
  Δ* = G*-E*      = 1.669574

恒等式（厳密に成立）:
  Δ*/Σ* = (S*-1)/(S*+1) = 0.372760  ✓

【δ_GERT の (Σ,Δ) 厳密表現】

GERTの定義: δ = S の変動幅
  S ∈ [S*-δ/2, S*+δ/2]

(Σ,Δ) での対応:
  S = (Σ+Δ)/(Σ-Δ) ↔ Δ/Σ = (S-1)/(S+1)

→ δ帯域 ⟺ (Σ,Δ)空間での Δ/Σ の変動域:
  Δ/Σ ∈ [(S*-δ/2-1)/(S*-δ/2+1), (S*+δ/2-1)/(S*+δ/2+1)]

線形近似（不動点周り）:
  δS = [Σ*·δΔ - Δ*·δΣ] / (2E*²)

  スケール係数: Σ*²/(2E*²) = 5.0835

  δS = 5.0835 · δ(Δ/Σ)

δ*_S = 0.2336 のとき:
  Δ/Σ ∈ [0.3489, 0.3949]   幅 = 0.0460
  中心 Δ*/Σ* = 0.3728

【IET × GERT 接続式（厳密版）】

GERTが制御するもの: S = (Σ+Δ)/(Σ-Δ)
IETが記述するもの:  分布 p(Σ, Δ)

接続:
  GERT の δ帯域
  ⟺ IET の p(Σ,Δ) における Δ/Σ の変動域
  ⟺ スケール係数 Σ*²/(2E*²) = 5.0835 で結ばれる

言葉で言うと:
  GERTが「S の帯域幅 δ を制御している」とは
  IETの言葉では「分布の Δ/Σ 比の変動域を制御している」こと

  S は (Σ,Δ) 平面上の「双曲線族の族パラメータ」であり
  δ は「その族の中で分布がどれだけの範囲を占めるか」

【IETが「帯域幅ではない」という直感の数学的根拠】

IETの自然な問いは:
  「どの (Σ*, Δ*) に中心を置くか」  ← μ の問い（r を選ぶ）
  「どれだけ σ を広げるか」           ← 揺らぎの問い（noiseを選ぶ）
  「Δ/Σ をいくつに保つか」           ← 比率の問い（a,bを選ぶ）

GERTの問いは:
  「S の変動を δ 以内に収めるか」    ← Δ/Σ 比の制御

→ IETの「中心」「形」「比率」という多次元の問いのうち
  GERTは「Δ/Σ の変動幅」という1つの射影だけを扱っている

これが「IETは帯域幅ではなく中心や分布形」という
直感の数学的実体である
