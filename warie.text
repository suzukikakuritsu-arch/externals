================================================================
☕ 4点の和 ≠ 0
η > 0 から直接
================================================================

【主張】

σ ∈ (0,1), u ∈ ℝ に対して:

ζ(σ+ui) + ζ(σ-ui) + ζ(1-σ+ui) + ζ(1-σ-ui) ≠ 0

【試み】

4点の和を計算する

ζ(σ+ui) + ζ(σ-ui)
= 2 Re(ζ(σ+ui))
= 2 Σ n^{-σ} cos(u log n)

ζ(1-σ+ui) + ζ(1-σ-ui)
= 2 Re(ζ(1-σ+ui))
= 2 Σ n^{-(1-σ)} cos(u log n)

4点の和:
= 2 Σ cos(u log n) [n^{-σ} + n^{-(1-σ)}]

これが 0 でないことを示す

================================================================
☕ η との接続
================================================================

η(σ) = Σ_{k=0}^∞ (1/(2k+1)^σ - 1/(2k+2)^σ)
     = Σ_{n=1}^∞ (-1)^{n+1} n^{-σ}
     > 0

4点の和:
F(u) = Σ cos(u log n) [n^{-σ} + n^{-(1-σ)}]

u = 0 のとき:
F(0) = Σ [n^{-σ} + n^{-(1-σ)}]
     = ζ(σ) + ζ(1-σ)
     < 0  （補題C）

F(0) ≠ 0 ✅

でも u ≠ 0 のとき F(u) = 0 になれるか？

F(u) = Σ cos(u log n) [n^{-σ} + n^{-(1-σ)}]

n^{-σ} + n^{-(1-σ)} > 0（各項正）
cos(u log n) は振動

F(u) = 0 になれる可能性がある

================================================================
☕ 止まる
================================================================

F(u) = 0 は u ≠ 0 で起きうる

例えば u が大きいとき
cos が激しく振動して
正と負が打ち消し合う

F(u) = 0 は零点の存在を
禁じない

================================================================
☕ 別の量を見る
================================================================

F(u)² + G(u)²

ここで:
G(u) = Σ sin(u log n) [n^{-σ} + n^{-(1-σ)}]

F(u)² + G(u)²
= |Σ e^{-iu log n} [n^{-σ} + n^{-(1-σ)}]|²
= |ζ_σ(u) + ζ_{1-σ}(u)|²

ここで ζ_σ(u) = ζ(σ+ui)

仮定: ζ_σ(u₀) = 0 かつ ζ_{1-σ}(u₀) = 0

→ F(u₀)² + G(u₀)² = 0
→ F(u₀) = 0 かつ G(u₀) = 0

でも:

F(u)² + G(u)²
= |Σ [n^{-σ} + n^{-(1-σ)}] e^{-iu log n}|²

= Σ_n Σ_m [n^{-σ}+n^{-(1-σ)}][m^{-σ}+m^{-(1-σ)}]
  × cos(u log(m/n))

n = m の対角項:
= Σ_n [n^{-σ}+n^{-(1-σ)}]²
> 0（各項正）

非対角項は正にも負にもなる

対角項だけで:
Σ_n [n^{-σ}+n^{-(1-σ)}]²
= Σ_n [n^{-2σ} + 2n^{-1} + n^{-2(1-σ)}]
= ζ(2σ) + 2ζ(1) + ζ(2(1-σ))

ζ(1) = ∞（発散）

→ 対角項の和が発散
→ F²+G² ≥ ∞？

これは形式的におかしい

================================================================
☕ 正しい下界
================================================================

Re(F²+G²) の下界:

Σ_n Σ_m [n^{-σ}+n^{-(1-σ)}][m^{-σ}+m^{-(1-σ)}]
× cos(u log(m/n))

≥ Σ_n [n^{-σ}+n^{-(1-σ)}]²  × cos(0)
  + Σ_{n≠m} ...（これが負になりうる）

下界が出ない

================================================================
☕ η で直接抑える
================================================================

η(σ+ui) = Σ (-1)^{n+1} n^{-σ} e^{-iu log n}

|η(σ+ui)|²
= η(σ+ui) × η(σ-ui)

= (Σ (-1)^{n+1} n^{-σ} e^{-iu log n})
  × (Σ (-1)^{m+1} m^{-σ} e^{iu log m})

= Σ_n Σ_m (-1)^{n+m} n^{-σ} m^{-σ} e^{iu log(m/n)}

n = m の対角項:
(-1)^{2n} n^{-2σ} = n^{-2σ} > 0

Σ_n n^{-2σ} = ζ(2σ)

σ ∈ (0,1/2) → 2σ ∈ (0,1)
ζ(2σ): 解析接続で有限かつ...

ζ(2σ) < 0 または > 0？

補題B: ζ(t) < 0 for t ∈ (0,1)
2σ ∈ (0,1) → ζ(2σ) < 0

→ 対角項の和 = ζ(2σ) < 0

でも対角項は全て正（n^{-2σ} > 0）
その和が負？

矛盾

ζ(2σ) = Σ n^{-2σ} は Re > 1 での話
2σ ∈ (0,1) では解析接続

解析接続された値は
Σ n^{-2σ} と等しくない
