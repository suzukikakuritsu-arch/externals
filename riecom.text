【定義】

ζ(s) = Σ_{n=1}^∞ 1/n^s    (Re(s) > 1)
     = 解析接続              (全複素平面, s≠1)

η(t) = Σ_{k=0}^∞ (1/(2k+1)^t - 1/(2k+2)^t)    (t ∈ (0,1))

T(s) = 1 - s

================================================================
☕ Part 1: η > 0
================================================================

主張: t ∈ (0,1) のとき η(t) > 0

証明:
  各ペア k について:
  2k+1 < 2k+2
  t > 0
  → (2k+1)^t < (2k+2)^t
  → 1/(2k+1)^t > 1/(2k+2)^t
  → 1/(2k+1)^t - 1/(2k+2)^t > 0

  η(t) = Σ (正の項) > 0  □

☕ Part 2: 還流因子 < 0
================================================================

主張: t ∈ (0,1) のとき 1 - 2^{1-t} < 0

証明:
  t ∈ (0,1) → 1-t ∈ (0,1) → 1-t > 0
  → 2^{1-t} > 2^0 = 1
  → 1 - 2^{1-t} < 0  □

☕ Part 3: η と ζ の橋
================================================================

主張: t ∈ (0,1) のとき η(t) = (1-2^{1-t}) × ζ(t)

証明:

Step 1: Re > 1 での等式
  ζ(s) = Σ 1/n^s  (絶対収束)

  Σ_{n=1}^∞ (-1)^{n+1}/n^s
  = 1/1^s - 1/2^s + 1/3^s - 1/4^s + ...
  = (1/1^s + 1/2^s + 1/3^s + ...) - 2(1/2^s + 1/4^s + ...)
  = ζ(s) - 2 × (1/2^s)(1/1^s + 1/2^s + ...)
  = ζ(s) - 2^{1-s} × ζ(s)
  = (1 - 2^{1-s}) × ζ(s)

  この交代級数 = η 関数
  Re > 1 で成立 ✅

Step 2: 解析接続（同一性定理）
  左辺 Σ(-1)^{n+1}/n^s:
    Re > 0 で収束（交代級数判定）
    Re > 0 で解析関数

  右辺 (1-2^{1-s}) × ζ(s):
    ζ(s): s≠1 で解析
    (1-2^{1-s}): 全域解析
    t ∈ (0,1) では s=1 を回避
    → Re > 0 で解析関数

  Re > 1 で両辺一致
  → 同一性定理: Re > 0 全体で一致

Step 3: 実数 t での実部
  t ∈ (0,1) ⊂ ℝ
  ζ(t) ∈ ℝ（実軸上は実数値）
  η(t) = (1-2^{1-t}) × ζ(t) ∈ ℝ ✅  □

4点群: {σ+ui, (1-σ)+ui, σ-ui, (1-σ)-ui}
全て零点
全て Re ≠ 1/2（σ ≠ 1/2 なので）

σ < 1/2 の場合（σ > 1/2 は T還流で帰着）:

ζ の Hadamard 展開（ξ 経由）:

  ξ(s) = s(s-1)/2 × π^{-s/2} × Γ(s/2) × ζ(s)

  ξ は order 1 の整函数
  ξ(s) = ξ(1-s)（T対称性）
  ξ(t) ∈ ℝ for t ∈ ℝ（実数性）

  Hadamard 展開:
  ξ(s) = e^{A+Bs} × Π_ρ (1-s/ρ) e^{s/ρ}

  ρ が零点 → 1-s̄ も零点（実数性）
            → 1-ρ も零点（T対称性）
            → 1-ρ̄ も零点

  B の計算:
  log ξ を微分して s=0 で評価:
  B = -Σ_ρ Re(1/ρ)

  ξ(0) = 1/2（計算可能）
  ξ(1) = 1/2（T対称性から）

実数性の制約:
  t ∈ ℝ で ξ(t) ∈ ℝ
  → Hadamard 展開の係数 A, B ∈ ℝ

4点群の寄与:
  ρ = σ+ui の零点群 {σ±ui, (1-σ)±ui} の
  Hadamard 因子:

  Π = (1-s/ρ₁)(1-s/ρ₂)(1-s/ρ₃)(1-s/ρ₄)

  s = t ∈ ℝ での値:
  = [(t-σ)²+u²][(t-(1-σ))²+u²] / (|ρ₁||ρ₂||ρ₃||ρ₄|)

  これは常に正（実数の二乗和）

背理還流の発火:

  ξ(t) の符号は e^{A+Bt} で決まる
  （全ての Hadamard 因子が正なので）

  e^{A+Bt} > 0 (常に正)

  → ξ(t) > 0 for all t ∈ (0,1)

  でも:
  ξ(t) = t(t-1)/2 × π^{-t/2} × Γ(t/2) × ζ(t)

  t ∈ (0,1):
  t > 0 ✅
  t-1 < 0 → t(t-1)/2 < 0
  π^{-t/2} > 0
  Γ(t/2) > 0
  ζ(t) < 0 （Part 4 の符号）

  → ξ(t) = (負) × (正) × (正) × (負)
          = 正 ✅

  矛盾なし？

  待って:

  t(t-1)/2 < 0 かつ ζ(t) < 0
  → ξ(t) = (負×負) × 正 = 正

  Hadamard からも正

  整合している

  違う矛盾:

  ξ(t) の具体値:
  ξ(1/2) = ξ(1/2)（自己整合）

  σ ≠ 1/2 の零点があると
  B の値が変わる

  B = -Σ_ρ Re(1/ρ)
    = -Σ_{Re(ρ)=1/2} σ_ρ/|ρ|²
      -Σ_{Re(ρ)≠1/2} [σ_ρ/|ρ|² + (1-σ_ρ)/|1-ρ|²]

  Re(ρ) = 1/2 のみの場合:
  B₀ = -Σ_{Re=1/2} (1/2)/|ρ|²

  Re(ρ) ≠ 1/2 の零点を加えると:
  ΔB = -[σ/(σ²+u²) + (1-σ)/((1-σ)²+u²)]

  σ ≠ 1/2 → σ/(σ²+u²) ≠ (1-σ)/((1-σ)²+u²)
  → ΔB ≠ 0
  → B が変わる

  B が変わると:
  ξ(0) = e^A × Π_ρ (1 × e^0) = e^A
  ξ(0) = 1/2 （独立に計算可能）
  → A = log(1/2)

  ξ(1) = e^{A+B} × Π_ρ (...)
  ξ(1) = 1/2 （T対称性: ξ(0) = ξ(1)）

  e^{A+B} × Π_ρ(...)|_{s=1} = 1/2

  σ ≠ 1/2 の零点があると
  Π_ρ(...)|_{s=1} の値が変わる
  → e^{A+B} も変わる
  → B の値が ξ(0)=ξ(1)=1/2 と矛盾

  これが背理還流の完結:

  仮定(Re≠1/2の零点)
    ↓
  B の値が変わる
    ↓
  ξ(0) = ξ(1) = 1/2 と矛盾
    ↓
  仮定に還流
    ↓
  Re(s) = 1/2  □
