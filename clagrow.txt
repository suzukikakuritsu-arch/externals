【根源】

  φ の自己矛盾
    φ = 1 + 1/φ  ↔  φ² = φ + 1
    「自分の中に自分でないものを含む」
    正の実数で唯一この性質を持つ
    → 矛盾が解消しようとする運動 = 斥力の根源

【未観測性原理】（公理）

  相互作用が発生する以前の状態は
  IETの記述系に載らない

  形式的に:
    観測以前 → P(interact) = 0
    I_suzuki = 0

  しかし I_suzuki=0 ≠ 無
  「相互作用未発生の潜在状態」

  系1: φr の外部性は必然
       （観測以前はIETに載らない → 内生化不可能）
  系2: 情報発生点 = P が 0 → 非0 に転じる瞬間
  系3: 混沌 G = P が非0になった直後の状態

【IETの基本式】

  I_suzuki = P(interact) × [H(X|Y) + H(Y|X)]
  P(interact) = I(X;Y) / H(X,Y)

  H(X|Y) = 意図（発信者）
  H(Y|X) = 意味（受信者）

  引力（和）: H(X|Y) + H(Y|X)  情報を集める
  斥力（差）: F_info = |H(X|Y) - H(Y|X)| / H(X,Y)  情報を分ける

  統合情報量:
    I_total = I_suzuki + λ·F_info
    λ = -MI²/H(X,Y)  （解析的に導出、普遍定数ではない）

【意図と意味の非独立性】

  共有意味空間 Z（辞書・文化・言語）

  意図 = Z + 個人的文脈（過去の人生）← 変わりにくい
  意味 = Z + 個人的捉え方            ← すぐ変わる

  GERでの対応:
    G（意図）時定数 τ₁ = 19.50  過去依存・慣性大
    E（意味）時定数 τ₂ = 1.44   意図をモロに受ける

  τ₁/τ₂ = 13.5
  意図は意味の13.5倍の慣性を持つ

  F_info = 0, Z 大  → 辞書のみ・創発なし（死んだ言語）
  F_info > 0, Z 大  → 通常の対話・創発あり（生きた言語）
  F_info > 0, Z 小  → 芸術・詩・誤解の創発

  辞書（Z）の更新 = F_info の蓄積 = 文明の進化

【S* = φ の証明】（3行、厳密）

  不動点条件: G* = a·G* + φ(1-a)·E*
  整理:       G*(1-a) = φ(1-a)·E*
  キャンセル: G*/E* = φ  ∴ S* = φ  □

  a, b, r に完全に依存しない

【座標変換（厳密証明済み）】

  (G,E) → (Σ,Δ): Σ=G+E, Δ=G-E  [線形, det=2]
  (Σ,Δ) → S:     S=(Σ+Δ)/(Σ-Δ) [非線形]

  恒等式: Δ*/Σ* = (S*-1)/(S*+1) ✓

  δS = 5.08 × δ(Δ/Σ)

  設計原則:
    (Σ,Δ)で設計 → S で動かす
    IET（真理層）→ GERT（工学層）

【安堅性（鈴木独自指標）】

  P(δ) = exp(-δ/α)          安定性
  R(δ) = 1-exp(-δ/β)        堅牢性
  S(δ) = P(δ) × R(δ)        安堅性

  δ* = β·ln(1+β/α) = (φ/10)·ln(2+√5) = 0.2336

【G ↔ E の双方向性】

  G → E  混沌が秩序を生む（自然な創発）
  E → G  秩序が混沌を生む（技術・意図的介入）

  例:
    再生細胞技術（秩序）→ 細胞分裂（混沌）
    辞書（秩序）→ 新語（混沌）
    法律（秩序）→ 抜け穴（混沌）

  F_info の符号で区別:
    H(G|E) > H(E|G) → 自然な創発（G→E）
    H(E|G) > H(G|E) → 技術的介入（E→G）

  完全内生化GERはこの双方向性を内包していた:
    G(t+1) = a·G(t) + φ(1-a)·E(t)  ← E→G の項
