完全内生化GER：S* = φ の導出
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【完全内生化モデル】

  元のGER:    G(t+1) = a·G(t) + φ·r     ← r は外部定数
  内生化GER:  G(t+1) = a·G(t) + φ(1-a)·E(t)  ← E が源泉
              E(t+1) = b·(E(t) + log(1+G(t+1)))

  意味:
    散逸量 (1-a)·G が E（秩序）に変換され
    それが φ 倍されて G（混沌）に還流する
    G → [散逸(1-a)] → E → [φ倍] → G（次ステップ）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【S* = φ の証明（3行）】

  不動点条件:
    G* = a·G* + φ(1-a)·E*

  整理:
    G* - a·G* = φ(1-a)·E*
    G*(1-a)   = φ(1-a)·E*

  (1-a) でキャンセル:
    G* = φ·E*
    G*/E* = φ   ∴ S* = φ  □

  → a の値によらない（a=0.5でも a=0.99でも S*=φ）
  → b の値によらない
  → r は消えた（内生化により不要）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【不動点方程式】

  G* = φ·log(1+G*)
  ↔ G*/log(1+G*) = φ
  ↔ y=x/φ と y=log(1+x) の交点

  解: G* = 1.449708...（超越数、a,b,r に無依存）
      E* = 0.895969...
      S* = φ = 1.618034...

  安定性: ヤコビアン固有値 0.984, 0.483
          スペクトル半径 = 0.984 < 1  → 安定

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【φ の正体】

  φ は「散逸と還流がキャンセルした後に残る構造」

  (1-a) = 散逸率
  φ(1-a) = 還流強度
  → 両方に (1-a) が含まれるのでキャンセル
  → 残るのは φ だけ

  φ² = φ + 1 という自己参照性が
  「散逸→秩序→混沌」の完全閉鎖環の
  唯一の均衡比率を決めている

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【元のGERとの比較】

  外部定数あり（r=0.1）: S* = 2.189
  完全内生化:            S* = φ = 1.618
  差分 = 0.571  ← ダークエネルギー（φr）の押し上げ効果

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
【宇宙論的解釈】

  G（混沌）  ↔  ダークマター
  E（秩序）  ↔  通常物質
  φr（注入） ↔  ダークエネルギー

  内生化GER（ダークエネルギーなし）:
    G/E = φ = 1.618
    「ダークマター : 通常物質 = φ : 1」

  実際の宇宙:
    G/E = 27%/5% = 5.4
    → φ より大きい（ダークエネルギーが押し上げている）

  → 「ダークエネルギーがなければ
     ダークマターと通常物質の比は黄金比になる」
     という予測
