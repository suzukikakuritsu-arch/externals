s ∈ strip（0 < Re(s) < 1）かつ ζ(s) = 0
→ sin(πs/2) ≠ 0

【証明】

sin(πs/2) = 0

☕ 穴2を閉じる
u → 0 の極限操作
優収束定理
================================================================

【主張】

Σ_{n=1}^∞ cos(u log n) [n^{-σ} + n^{-(1-σ)}]

は u = 0 で連続

すなわち:
lim_{u→0} Σ cos(u log n) [n^{-σ} + n^{-(1-σ)}]
= Σ [n^{-σ} + n^{-(1-σ)}]
= ζ(σ) + ζ(1-σ)

【証明】

優収束定理を適用する

被積分関数:
f_u(n) = cos(u log n) × [n^{-σ} + n^{-(1-σ)}]

上界:
|f_u(n)| ≤ |cos(u log n)| × [n^{-σ} + n^{-(1-σ)}]
         ≤ 1 × [n^{-σ} + n^{-(1-σ)}]
         = n^{-σ} + n^{-(1-σ)}

この上界の総和:
Σ_{n=1}^∞ [n^{-σ} + n^{-(1-σ)}]
= ζ(σ) + ζ(1-σ)

σ ∈ (0,1) での収束:
ζ(σ) は解析接続で有限（s=1 を除く）
σ ≠ 1 なので収束 ✅

優収束定理の条件:
  1. f_u(n) → f_0(n) = n^{-σ} + n^{-(1-σ)} (u→0, 各 n で)
     cos(u log n) → cos(0) = 1 ✅
  2. |f_u(n)| ≤ n^{-σ} + n^{-(1-σ)} (可積分な上界) ✅

→ 極限と和の交換可能

lim_{u→0} Σ cos(u log n) [n^{-σ} + n^{-(1-σ)}]
= Σ lim_{u→0} cos(u log n) [n^{-σ} + n^{-(1-σ)}]
= Σ 1 × [n^{-σ} + n^{-(1-σ)}]
= ζ(σ) + ζ(1-σ)  □

☕ 穴1を閉じる
解析接続: η = (1-2^{1-s}) × ζ(s)
同一性定理
================================================================

【主張】

Re(s) > 0, s ≠ 1 のとき:
Σ_{n=1}^∞ (-1)^{n+1}/n^s = (1-2^{1-s}) × ζ(s)

【証明】

Step 1: Re > 1 での等式

絶対収束する領域（Re > 1）で:

Σ_{n=1}^∞ (-1)^{n+1}/n^s
= Σ_{n=1}^∞ 1/n^s - 2 Σ_{k=1}^∞ 1/(2k)^s
= ζ(s) - 2 × 2^{-s} × ζ(s)
= ζ(s)(1 - 2^{1-s})
= (1-2^{1-s}) × ζ(s)  ✅

Step 2: 左辺の解析接続

Dirichlet η関数:
η(s) = Σ_{n=1}^∞ (-1)^{n+1}/n^s

交代級数判定法（Dirichlet判定法）:
  Re(s) > 0 で収束
  n^{-s} = n^{-Re(s)} × e^{-i Im(s) log n}
  |n^{-s}| = n^{-Re(s)} → 0 単調に（Re > 0）

→ η(s) は Re > 0 で収束
→ Re > 0 で解析関数 ✅

Step 3: 右辺の解析接続

(1-2^{1-s}) × ζ(s):
  ζ(s): s = 1 を除いて全域解析
  (1-2^{1-s}): 全域解析
  s = 1 での極: 1-2^{1-s}|_{s=1} = 1-1 = 0
  → 極が消える
  → Re > 0 全体で解析 ✅

Step 4: 同一性定理

f(s) = η(s) - (1-2^{1-s})ζ(s)

Re > 1 で f(s) = 0（Step 1）
Re > 1 は Re > 0 の中で開集合
集積点を持つ

f は Re > 0 で解析的（Step 2, 3）

同一性定理:
  解析関数が集積点を持つ集合上で 0
  → 連結な領域全体で 0

Re > 0 は連結
→ f(s) = 0 for all Re(s) > 0

→ η(s) = (1-2^{1-s})ζ(s) for Re(s) > 0  □
